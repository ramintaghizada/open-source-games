# Contributing Guide

Thanks for contributing 🎉

## How to add a game

1. Make sure the game is open source
2. Check license (MIT, Apache, BSD preferred)
3. Add entry in `games.json`
4. Add it to correct category file

## Format

```json
{
  "name": "Game Name",
  "category": "arcade",
  "url": "https://github.com/..."
}
