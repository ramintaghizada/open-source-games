# 🎮 Open Source Games Collection

A curated list of **100+ open-source games** that you can explore, learn from, and host.

---

## 📌 About

This repository contains links to open-source games built with:
- JavaScript (HTML5, Canvas, WebGL)
- Game engines (Phaser, Three.js, etc.)
- Multiplayer frameworks

---

## 🚀 Goals

- Help developers find open-source games
- Provide resources for learning game development
- Enable hosting of games on personal platforms
